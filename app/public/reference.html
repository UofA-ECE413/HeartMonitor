References

Third-party APIs, libraries, and code.

Reference Description and Usage
JQuery A JavaScript library that makes things like HTML document traversal and manipulation, event handling, animation, and Ajax much simpler with an easy-to-use API that works across a multitude of browsers. Used to make JavaScript code simpler and more readable.
JQuery Validation A JQuery plugin that simplifies client-side form validation. Used to ensure that forms are filled out correctly and completely and provide users with insights on invalid forms.
Chart.js An open source library for visualizing data and creating charts. Used to visualize daily data for heart rate and pulse oxygen levels.
MAX30105 Particle Library A Particle Community Library for MAX30105 and similar sensors. Used to collect heart rate and pulse oxygen data from MAX30102 sensors to provide insights for patients and users.
